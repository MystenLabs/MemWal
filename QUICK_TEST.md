# Quick Test - v0.3.4 🚀

## Restart Dev Server

In your terminal where `p dev` is running:
1. Press `Ctrl+C` to stop it
2. Run: `pnpm run dev`
3. Wait for "Ready in Xms"

## Test It!

Visit: **http://localhost:3000/showcase**

### Say this:
```
"My name is Zan and I love chicken and badminton"
```

