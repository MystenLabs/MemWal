## Features

- **Encrypted Storage**: SEAL encryption, persisted to Walrus blobs
- **Semantic Recall**: Embedding search for relevant memories in milliseconds
- **Delegate Keys**: Low-risk keys for apps, revocable anytime onchain
- **AI Middleware**: Wrap models with memory context using one SDK
