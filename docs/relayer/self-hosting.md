# Self-Hosting

Self-host the relayer when you need operational control over the backend service.

## Why Self-Host

- control your own deployment environment
- manage your own database and secrets
- tune operational behavior for your application

## What You Need

- PostgreSQL with pgvector
- Sui RPC access
- Walrus publisher and aggregator endpoints
- embedding provider credentials
- server wallet configuration for SEAL-related flows
